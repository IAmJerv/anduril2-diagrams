# Anduril 2 UI Diagrams
User Interface digrams for Anduril 2 - an advanced flashlight UI developed by ToyKeeper

## Background
This was 100% inspired by [Lux-Perpetua’s Andúril 2 UI diagram on BLF](https://budgetlightforum.com/node/76941). In fact, Lux’s diagram is so good, that I would have been happy to just use it except for one thing: the source isn’t published, so you can’t modify it. I also wanted to be able to keep it up to date with [ToyKeeper’s code](https://github.com/ToyKeeper/anduril). So I created my own…
